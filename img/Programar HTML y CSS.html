🔍 PRINCIPALES CAMBIOS

  1. Centrado general con .equipo y .grupo.

  2. .integrante ahora es columna, no fila, para que imagen y texto estén uno arriba del otro.

  5. Se usa gap para espaciado uniforme entre perfiles.

🔍 Detalles

  • rem se basa en el tamaño de fuente raíz (por defecto 1rem ≈ 16px).

  • % usado en max-width: 90% permite que el contenido no se expanda más allá del ancho de pantalla.

  • gap, margin, font-size, y max-width ahora son totalmente escalables, ideales para diseño responsive.

¿Querés que también se adapte a pantallas móviles? Puedo agregar un @media para eso.
